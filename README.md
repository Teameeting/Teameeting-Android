### Teameeting 简介
一款用集成多人视频通话，群聊功能应用，多达4同时实现通话，多人群聊，同时拥有**Android - IOS - Web- TV盒子**全平台所有版本，轻松实现**全平台**视频通话以及聊天功能。**开放所有源码**，**音频清晰**，断点续传，自适应网速优先音频传输；

### 项目功能
1. 高清多人视频通话；
2. 多人群聊语音，群聊天；
3. 微信分享点击链接即可加入视频通话；
4. 消息实时提醒；
5. 无需注册；
6. 极光推送群聊，视频邀请时时提醒；
7. 离线消息保持；
8. 加入聊天室列表自动保存；
9. 聊天室权限控制以及聊天室控制；
10. 适配平板以及天猫电视盒子；
